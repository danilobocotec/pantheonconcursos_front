# Exemplo de uso do Editor Rich Text

## Funcionalidades implementadas:

### 1. Formatação de texto
- **Negrito**: Botão com ícone de negrito
- *Itálico*: Botão com ícone de itálico

### 2. Alinhamento
- Esquerda
- Centro
- Direita

### 3. Listas
- Lista com marcadores (bullet list)
- Lista numerada (ordered list)

### 4. Mídia
- **Imagem**: Clique no ícone de imagem e insira a URL
- **Vídeo**: Clique no ícone de vídeo e insira URL do YouTube
- **Áudio**: Clique no ícone de música e insira URL do áudio (cria um link clicável)

## Como usar:

1. Acesse Admin Dashboard
2. Vá em "Cursos" > "Itens"
3. Clique em "Novo item" ou "Editar" em um item existente
4. No campo "Conteudo", você verá a barra de ferramentas com todos os botões
5. Use os botões para formatar o texto
